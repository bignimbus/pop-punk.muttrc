## Installation

Copy `pop-punk.muttrc` from this repository to your `~/.mutt` directory.
In `.muttrc`, add:

```sh
source ~/.mutt/pop-punk.muttrc
```

The code for this theme owes its structure to the
[Dracula](https://draculatheme.com/mutt/) theme.
